#UserFormAppMVC

這是一個使用 **ASP.NET Core MVC 開發的簡易會員系統專案，
目的是學習 MVC 架構、表單驗證、Session 管理與圖片上傳等功能。

---

#技術架構

ASP.NET Core MVC (.NET 6+)
Razor View Engine
C# 語言
HTML / CSS / Bootstrap 5
Session 登入驗證
密碼加密（使用 `BCrypt.Net`）
圖片上傳與檔案存取

---

#功能特色

會員註冊（含驗證）
登入 / 登出
密碼加密儲存（使用 BCrypt）
表單驗證（必填、Email 格式、年齡範圍）
照片上傳並顯示於資料表格
Session 管理（未登入不能瀏覽會員頁面）
